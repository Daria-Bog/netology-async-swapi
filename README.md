# Async SWAPI Loader

Асинхронное приложение на Python для выгрузки данных о персонажах вселенной Star Wars из [SWAPI](https://www.swapi.tech/) и их сохранения в базу данных PostgreSQL.

## Технологии
* **Python 3.11+**
* **Asyncio / aiohttp** — для асинхронных HTTP-запросов.
* **SQLAlchemy (Async)** — для работы с базой данных.
* **PostgreSQL** — в качестве хранилища данных.
* **Docker Compose** — для быстрого развертывания базы данных.

## Описание работы

Приложение реализовано с использованием концепции асинхронного программирования для обеспечения высокой производительности:

* **Параллельная выгрузка:** Скрипт разделяет общий список ID персонажей на порции (чанками) по 10 штук. Запросы к API выполняются конкурентно с помощью `asyncio.gather`.
* **Неблокирующая запись:** Сохранение данных в базу данных PostgreSQL выносится в отдельные фоновые задачи (`asyncio.create_task`). Это позволяет основной программе приступать к загрузке следующей пачки данных, не дожидаясь завершения транзакции в БД.
* **Корректное завершение:** В конце работы скрипт ожидает завершения всех фоновых задач записи и закрывает пул соединений с базой данных через `engine.dispose()`.
